# Steps to clone and use the project  

1. Python version must be 3.12.xx  
  `python --version`

2. Clone the repository  
  `git clone https://github.com/abhayx3324/digicom.git`

3. Open digicom directory  
  `cd digicom`

4. Create a new virtual environment  
`pythom -m venv venv`

5. Install dependencies from requirements.txt file  
`pip install -r requirements.txt`

6. Run the application  
`python main.py`
